1. Установка
1.1 Установите среду выполнения python3.7
1.2 Разверните новое виртуальное окружение с помощью venv
1.3 Активируйте добавленное окружение
1.4 Подтяните зависимости в окружение с помощью pip (файл requirements.txt)

2. Использование
2.1 Активируйте созданное окружение
2.2 Запустите main.py
2.3 Откройте в браузере http://localhost:8000/
2.4 Тестируйте
*первый запрос проходит до 30 секунд, последующие - в районе 7 секунд

3. Обучение (если необходимо поиграть с гепотизами обучения или учить на новом корпусе)
3.1 Скачайте данные - не включил в проект базу на 3.5Гб с твитами, ее скачать можно https://study.mokoron.com/, а затем импортните ее в sqlite с помощью https://github.com/dumblob/mysql2sqlite, положите как data/mysqlite3.db*
3.2 Обучите модель представления слов w2v - выполните study/fit_w2v.py
3.3 Обучите модель с замороженным эмбендингом - study/fit_CNN_withfrozenembending.py
3.4 Выберите модель с лучшей метрикой (по названию сохраненных файлов [эпоха]-[метрика])
3.5 Укажите путь к ней в тексте study/fit_CNN.py и запустите его
3.6 Сохраните модель для быстрой загрузки через веб - запустите study/savemodel.py 
*нужно для обучения w2v модели, если просто меняете параметры сетки и переучиваете - не надо, если меняете корпус текстов - соответсвенно нужен он + переписать скрипты считывания данных (см. study/getdata.py), а также заменить csv с размеченными данными - в папке data
